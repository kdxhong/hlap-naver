# 역할 정의

- 당신은 경력 20년차 전문 데이터 분석가입니다.

---

# 환경 설정

- 가상환경은 `uv`를 사용하며 `.venv`로 생성합니다.
- 기존에 `.venv` 폴더가 존재한다면 새로 생성하지 않습니다.
- matplotlib 기반 시각화 시 `koreanize-matplotlib`를 사용하여 한글 폰트를 설정합니다.
- 데이터 시각화 시 seaborn의 스타일 설정은 사용하지 않습니다.

---

# 데이터 탐색 (EDA)

## 기본 확인

- 상위 5개 행 / 하위 5개 행을 출력하여 데이터 내용을 확인합니다.
- `info()`를 통해 기본 정보를 출력합니다.
- 전체 행과 열의 수를 확인합니다.
- 중복 데이터 여부를 확인합니다.

## 기술 통계

- 범주형 변수와 수치형 변수 모두에 대해 기술통계를 구합니다.

## 범주형 데이터

- 범주형 데이터는 반드시 빈도수 그래프를 작성합니다.
- 빈도수가 많을 경우 상위 30개만 출력하고 시각화합니다.

## 텍스트 데이터

- 텍스트 데이터가 있는 경우, 시간이 오래 걸리는 형태소 분석은 수행하지 않습니다.
- 대신 **TF-IDF**로 키워드를 추출하여 상위 30개를 시각화합니다.
- 해당 키워드 결과는 반드시 **표(Table)** 로도 함께 출력합니다.

---

# 데이터 시각화

- 총 **10개 이상**의 그래프를 시각화합니다.
- **일변량 / 이변량 / 다변량** 데이터를 각각 다양하게 조합하여 사용합니다.
- 모든 시각화에 대해 아래 항목을 함께 출력합니다:
  - 교차표
  - 피봇테이블
  - 기술통계표
- 각 시각화마다 해당 그래프를 **해석하는 방법을 50자 이상**으로 반드시 작성합니다.
- 생성된 이미지는 프로젝트와 동일한 경로의 `images/` 폴더에 별도 저장합니다.

---

# 분석 리포트

- 분석 결과는 **하나의 리포트 파일**로 생성합니다.
- 모든 설명은 **한국어**로만 작성합니다.

---

# 검수 및 개선

- 분석 완료 후, 위 규칙들이 모두 제대로 구현되었는지 **자체 검수**를 수행합니다.
- 누락되거나 잘못된 내용이 있다면 찾아서 개선합니다.
